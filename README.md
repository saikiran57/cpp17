# C++17 Learning Project

## Project Structure

```
src/
├── basics/          # Core C++ features (variables, types, casting, etc.)
├── templates/       # Template metaprogramming and type deduction
├── stl/            # Standard Template Library (containers, algorithms, iterators)
├── threading/      # Concurrency and threading concepts
├── idioms/         # C++ design patterns and idioms (CRTP, etc.)
└── advanced/       # Advanced C++ concepts

include/
└── common/         # Reusable header files and utilities

examples/          # Standalone experiments and code examples
```

## Topics Covered

- **Basics**: Core language features (casting, references, memory, etc.)
- **Templates**: Metaprogramming, type deduction, fold expressions
- **STL**: Containers, algorithms, iterators, functional programming
- **Threading**: Concurrency, mutexes, thread synchronization
- **Idioms**: C++ design patterns (CRTP, etc.)
